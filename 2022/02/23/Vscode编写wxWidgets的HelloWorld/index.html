Vscode编写wxWidgets的HelloWorld

发表于2022-02-23|更新于2024-02-19|vscodewxwidgetsc/c++
|阅读量:

软件及资源包

  1. 下载并安装Vscode代码编辑器
    软件安装后还需在vscode的插件管理中安装C/C++开发插件
  2. 下载并安装MinGW-w64Windows下的GNU编译器
    下载编译器的版一定跟后面下载的wxWidgets的软件包编译使用的版本保持一致
  3. 从wxWidgets下载下图几压缩包
    资源包截图 Header Files是wxWidgets的头文件,Development Files是编译所需的一些库文件件和动态库,Release DLLs是编译后的软件，发布给用户时需一起打包发送。不然用户有可能会报缺少dll文件的错误

软件包安装

wxWidgets安装

将上面下载的wxWidgets头文件和开发包，都解压到同一个文件夹下(这里我们以D:\wxWidgets_3.1.5为例)。解压后目录结构如下：

将D:\wxWidgets_3.1.5\lib\gcc810_dll添加到环境变量path中，方便生成的应用程序正接运行。

MinGW-w64安装

将上面下工的i686-5.4.0-release-win32-dwarf-rt_v5-rev0.7z解压到D:\mingw-64\i686-8.1.0-release-win32-sjlj-rt_v6-rev0目录下。

将D:\mingw-64\i686-8.1.0-release-win32-sjlj-rt_v6-rev0\mingw32\bin添加到环境变量path中，这样在终端就可以直接使用MinGW的命令了，vscode也可以自动检测了编译器路径。

Vscode编译文件

1. 创建Hello_wxWidgets.cpp

新建一个工程文件夹，使用vscode打开目录。在vscode中新建一个Hello_wxWidgets.cpp文件,并粘贴如下代码：

Hello_wxWidgets.cpp

2. 配置Vscode的C/C++编辑属性

在vscode中按 F1 或 Ctrl+Shift+P 输入:c/c++: edit configurations (ui)将会自在.vscode目录下生成一个c_cpp_properties.json文件。:

c_cpp_properties.json

注意：

  1. 如果是vc编译就需要添加D:\wxWidgets_3.1.5\include\msvc
  2. 如果是gcc编译就需要添加D:\wxWidgets_3.1.5\lib\gcc810_dll\mswud
  3. gcc810_dll\mswud对应debug头文件，gcc810_dll\mswu对应release头文件

3. 配置编译任务

在vscode中按 F1 或 Ctrl+Shift+P 输入:tasks: configure task如下图：

tasks.json
  1. -ID:\\wxwidgets_3.1.5\\include和-ID:\\wxWidgets_3.1.5\\lib\\gcc810_dll\\mswud添加头文件目录，编译的时候会从这些目录中去找头文件。
  2. -LD:\\wxWidgets_3.1.5\\lib\\gcc810_dll添加库文件目录，链接的时修改会从这些目录中去找库文件。
  3. -lwxbase31ud和-lwxmsw31ud_core是链接时引用的库文件。
  4. -mwindows表示生成windows应用程序，不配置的话启动程序会有一个黑框出现。
  5. -pipe表示编译时使用管道
  6. -Wall表示编译时输出全部警告

可以看到编译生成的Hello_wxWidgets.exe成功运行起来了。
